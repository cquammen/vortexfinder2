# README #

### What is this repository for? ###

* Vortex detection for condor2/GLGPU written in C++
* Version 0.1

### Prerequisites ###

* CMake (version >= 3.0.2)
* PETSc (version >= 3.5.2)
* libMesh (preferable 0.9.4-rc1) built with PETSc
* Qt4 (optional for the viewer, preferable 4.8.6)
